# Password Validator

## Summary

This project is a Password Validator built using React Hooks and Styled Components.  
It validates whether a password is at least 8 characters long and displays an error message when the password is invalid.

---

## Project Description

The Password Validator app checks the length of the password entered by the user.

- If the password length is less than 8 characters, an error message is shown
- If the password length is 8 or more characters, the error message is hidden

---

## Functionality

- When the app loads, the error message is displayed
- When the input is empty or contains fewer than 8 characters, the error message is displayed
- When the input contains 8 or more characters, the error message is hidden
